# 2DGame
2D game using OpenGL

# Controls
- W - move forward
- A - move left
- S - move back
- D - move right

# Rules 
- Collect coins to increase score!
- Avoid the enemy blocks 
- Reach the gate at the end to progress to next level

# Instruction to run
```
mkdir build
cd build
cmake .. 
cd ..
build/game
```
